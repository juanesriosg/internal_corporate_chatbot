Identity Claims

The chatbot receives OIDC claims for subject, email, tenant_id, department, groups, and manager_id. The backend must verify token issuer, audience, expiration, and signature before using claims.

ACL Enforcement

Document chunks must include allowed_groups, tenant_id, sensitivity, and source_acl_hash. Retrieval must apply metadata filters before prompt construction. The LLM must never receive unauthorized chunks.

Cache Safety

Answer caches must include tenant_id, user group hash, query hash, and document ACL hash. Shared caches without ACL-aware keys are prohibited.

Audit Events

The backend records query_id, user_id hash, retrieved chunk IDs, answer ID, model ID, and access-filter decisions. Raw user questions are retained for 30 days in the pilot.
