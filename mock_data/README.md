# Northstar Digital Mock Corporate Knowledge Base

This zip contains 20 mock internal documents for testing an internal corporate RAG chatbot.

## Formats included

- PDF
- Microsoft Word `.docx`
- Markdown `.md`
- HTML `.html`
- Plain text notes `.txt`

## Testing goals

- Multi-format parsing
- Chunking with headings and metadata
- Source attribution and citations
- Department and group-based access control
- Stale-document conflict handling
- Prompt-injection resistance
- Restricted document filtering

## Important

All content is fictional. Do not treat these documents as real policies.

See `manifest.json` for document metadata, recommended ACL tests, and sample questions.
